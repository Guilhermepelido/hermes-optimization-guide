## 🚀 Setting up the Hermes Agent
After installation, open the application from your desktop shortcut or the Start menu. The first time you launch the program, the system asks for your preferences. 

### Configuration options
- **Agent Name:** Choose a label for your identification.
- **Data Folder:** Select where the program stores your migration files.
- **Telegram Integration:** Keep your credentials ready if you want real-time alerts.

The agent runs in the background. You can access the main dashboard at any time by clicking the icon in your system tray, located near the time and volume settings on your taskbar.

## 📚 Using LightRAG for data management
LightRAG functions as the memory center for your agent. It allows the software to retrieve information efficiently. To activate this feature:
1. Open the "Settings" menu inside the dashboard.
2. Select the "LightRAG" tab.
3. Toggle the switch to "Enabled".
4. Choose the library folder where you keep your documents.
5. Save your changes and restart the application to apply the settings.

The agent now indexes your documents automatically. This process takes a few minutes depending on the volume of files in your folder.

## 📱 Telegram notifications
You can receive alerts directly on your phone. This keeps you informed about agent performance while you are away from your desk.
1. Create a bot through the Telegram BotFather.
2. Copy the API token provided by the BotFather.
3. Paste this token into the "Telegram" section of your agent settings.
4. Send a test message to verify the connection is stable.

## 🛠️ Building custom skills
Skills define what the Hermes Agent does for you. You do not need to be a developer to add functionality. 
1. Open the "Skills" tab in the application.
2. Select "Create New Skill".
3. Assign a name and a trigger task.
4. Define the conditions that launch your skill.
5. Click "Save" to add the skill to your library.

You can modify these skills whenever your requirements change. The interface handles the internal logic, so you only focus on the task objectives.

## 🔄 Migrating existing data
If you used previous versions of agent software, you can bring that data into this system using the migration tool:
1. Go to "Tools" and select "Migration Wizard".
2. Point the wizard to your old data directory.
3. The application scans the files and reformats them for your current installation.
4. Confirm the transfer and wait for the summary report.

Always perform a backup of your old folder before starting a migration. This protects your data in case of unexpected errors during the move.

## ❓ Troubleshooting common issues
If you encounter difficulty, check these common fixes:
- **Application does not start:** Verify you have administrative rights on your machine.
- **Telegram triggers fail:** Confirm your API token is copied correctly without extra spaces.
- **High memory usage:** Limit the number of documents in your LightRAG folder to improve performance.
- **Dashboard disappears:** Check your system tray icons, as the window may be minimized, not closed.

If issues persist, delete the temporary log files located in your installation folder and restart the application. This clears caches that might cause slow performance.
